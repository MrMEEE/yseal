Getting Started

1

Install the CLI

pip install yseal-cli

Install the ySEal command-line tool to browse and download policies.

2

Search for Policies

yseal search docker

Search for SELinux policies by keyword, tag, or contributor.

3

Install Policies

yseal policy install contributor.policy

Download and install SELinux policies to your system.
